🔥 Firebase real-time setup

Follow these steps once — takes ~3 minutes. Free tier is more than enough.

  1. Go to console.firebase.google.com → Add project → give it any name → disable Google Analytics → Create project.
  2. Inside your project → Build → Firestore Database → Create database → choose Start in test mode → pick any region → Enable.
  3. Go to Project settings (gear icon) → General tab → scroll to "Your apps" → click the </> Web icon → register app with any nickname → copy the firebaseConfig object values below.
  4. Paste each value into the fields below and click Save & connect.
  5. Commit & push index.html to your GitHub repo — everyone who opens the page will now see live updates instantly.
